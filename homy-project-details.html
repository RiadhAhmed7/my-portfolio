IoT Project

Homy - Smart Home Automation

Project Overview

The Homy Smart Home Automation system is an innovative IoT solution that allows users to control various home appliances remotely. The system provides real-time monitoring of energy consumption and environmental conditions.

Key Features

  • Remote control of lights, TV, and air conditioning
  • Real-time energy consumption monitoring
  • Temperature and humidity monitoring with DHT11 sensor
  • Gas leak detection system
  • Mobile app for iOS and Android
  • Energy usage reports and analytics

Technical Implementation

The system uses ESP32 microcontrollers connected to various sensors and relays. Data is transmitted via MQTT protocol to a cloud-based server. The mobile app, developed with Flutter, provides a user-friendly interface to control devices and view statistics.

Technologies Used

ESP32 NodeMCU DHT11 Sensor MQTT Protocol Firebase Flutter Arduino IDE Raspberry Pi

Results & Impact

The Homy system successfully reduced energy consumption by 25% in test environments by providing real-time usage data and automated scheduling. Users reported a 40% increase in convenience and home security.
